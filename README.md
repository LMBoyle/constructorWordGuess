# Constructor Word Guess
A simple, web-developer, word guess game.


## Instructions
Use in a terminal.

Press letters on your keyboard to make a guess. Good luck!

## What It Teaches
How to use node.js with constructor functions

## Languages/Technologies 
* JavaScript
* Node.js

### NPM Packages
